折腾Hugo终于好了

2021-06-06
深海电音咒语

因为 Gridea 这两天一直连接错误，于是打算转到别的生成静态博客的平台。看了下网上的评论，发现 Hugo 会比 Hexo 容易折腾点所以就选了 Hugo ，然后正式开始折腾。

经过一天一夜的折腾，终于把 Blog 搬到 Hugo 了。

一开始挑选的主题一直 error 了几个小时，到下午 push 到 Github 一直错误……鬼知道我经历了什么。

虽然一直在来回翻译和找中文资料，但功夫不负有心人，我终于搞好了，成就感爆棚！

如果后面有时间会整理一下折腾 Hugo 的一系列教程？
