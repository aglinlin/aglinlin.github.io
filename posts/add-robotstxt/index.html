创建 robots.txt

2023-10-03
深海电音咒语

晚上突然发现自己的域名被污染了，就很郁闷……半个月前看 Umami 发现有来自 Baidu 的访问，隐约觉得不安。虽然不知道是不是 Baidu 的锅，但总之紧急给博客加个禁止机器人抓取信息。按官方教程 Robots.txt file | Hugo 添加了 robots.txt。不知道有没有用，但总之，加上！

在配置文件加上：

enableRobotsTXT = true

接着在 layouts 文件夹下新增文件 robots.txt，内容如下：

User-agent: *
{{ range .Pages }}
Disallow: {{ .RelPermalink }}
{{ end }}

按官方教程说明这样就是禁止所有机器人抓取所有页面信息。

唉，我恨！

10/04
破案了，看了一下 TG 说是大面积 DNS 污染加劫持，许多博客都打不开了。跟着爱海的 最新博文 把名称服务器从 Vercel 迁到 Cloudflare 了。
